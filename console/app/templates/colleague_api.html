Colleague Web API

Call Colleague CTX transactions and inspect event configurations via the Ellucian Colleague Web API (Basic auth).

Connection required. Add COLLEAGUE_WEB_API_URL and credentials to .env to enable this feature.
Setup steps
  1. Add to .env:
COLLEAGUE_WEB_API_URL=https://colleague-webapi.doane.edu
COLLEAGUE_WEB_API_USER=your-service-account
COLLEAGUE_WEB_API_PASS=your-password
  1. Restart the console — this tab activates automatically.
Supports CTX transaction calls (POST /api/transactions/{id}), event configuration review, and API metadata browsing.
