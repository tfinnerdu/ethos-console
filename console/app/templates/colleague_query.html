Colleague Direct Query

Run UniQuery SELECT statements directly against Colleague files via ODBC. Useful for spot-checking data during Ethos integration development without opening a full Colleague session.

Connection required. Add UNIDATA_HOST and related vars to .env to enable this feature.
Setup steps
  1. Add the following to .env:
UNIDATA_HOST=your-unidata-server
UNIDATA_PORT=31438
UNIDATA_USER=your-service-account
UNIDATA_PASSWORD=your-password
UNIDATA_ACCOUNT=D:\Ellucian\coll18\test\apphome
  1. Ensure uopy==1.4.0 is in requirements.txt and reinstall.
  2. Restart the console — this page will activate automatically.
Uses Rocket Software's uopy API over port 31438 (U2 OIPC). Read-only — run TCL commands like LIST VOC or LIST PERSON SAMPLE 5.
